# Workshop Overview

## Learning Flow
**Build → Orchestrate → Evaluate**

1. **Build (Lab 1):** Create a persistent agent in the Foundry portal.
2. **Orchestrate (Lab 2):** Reuse that agent in Python and orchestrate a multi-agent workflow.
3. **Evaluate (Lab 3):** Run repeatable evaluations to establish quality baselines before production.

## What learners will produce
- A persistent Foundry agent: `ProductInventoryManager`
- A second agent created via SDK (Lab 2): `SalesInsightsAgent`
- A small evaluation dataset and evaluation results (Lab 3)
